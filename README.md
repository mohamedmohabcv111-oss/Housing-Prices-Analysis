# Housing-Prices-Analysis

# Highlights
Cleaned and engineered features from a large real-estate dataset.

Exploratory analysis with clear visual summaries by bedrooms, bathrooms, and price.

Predictive modeling: baseline linear regression and a Random Forest (OOS R² ≈ 0.60 on log-price).

Interactive Shiny dashboard for KPI inspection and filtering.

Tech & tools R, tidyverse, ggplot2, ranger, Quarto, Shiny

# Quick start
Clone repo and place the dataset in data/.

Open housing_Pricing_Final.qmd in RStudio and render with Quarto.

Run the Shiny section to launch the interactive app.
